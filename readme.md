Nama Proyek: PerpusDigital  
Deskripsi: Website perpustakaan digital sekolah yang memungkinkan pengguna untuk login, mendaftar, meminjam, dan mengembalikan buku secara online.

Fitur utama

- Login dan Register pengguna
- Tambah, lihat, dan hapus buku
- Peminjaman dan pengembalian buku
- Admin dapat mengedit data buku
- Riwayat transaksi pengguna

Teknologi
HTML, CSS, JavaScript, PHP, MySQL

Struktur folder
/assets – berisi gambar, ikon, dan file CSS/JS  
/auth – berisi halaman login dan register  
/pages_admin – halaman admin  
/pages - halaman user
index-admin.html – dashboard admin  
index.html – halaman utama pengguna

Cara mendoploy secara lokal

1. Jalankan server lokal (misal XAMPP / Laragon).
2. Simpan folder project di htdocs.
3. Import database MySQL (perpusdigital.sql).
4. Buka browser dan ketik http://localhost/perpusdigital/
